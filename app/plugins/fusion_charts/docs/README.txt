##########################################
# Fusion Charts Plugin for CakePHP v.0.1 #
##########################################

Thank you for trying out Fusion Charts plugin for CakePHP!

Please note that this is only an initial release of a work-in-progress plugin,
with only the basic funcionality to reproduce the charts available in the
official FS Free zip file. More features will be added and the existing ones
will be improved as you people start to complain.

For a list of changes in this release, please read the CHANGELOG file.

Please send your comments and suggestions to <neutrinocms[at]gmail.com>
